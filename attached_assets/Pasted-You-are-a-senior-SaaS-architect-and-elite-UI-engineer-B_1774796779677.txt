## 🔌 CORE FEATURES

### 1. Email Detection API

Route: `/api/check-email`

* Input: email

* Extract domain

* Check against Domain table

* Return:
  {
  isDisposable: boolean,
  domain: string
  }

* Must be optimized:

  * Load domains into memory cache on server start
  * Fallback to DB query if needed

---

### 2. Domain Sync System

* Fetch domain list daily from GitHub
* Store in PostgreSQL
* Remove duplicates
* Normalize domains

Admin feature:

* “Sync Now” button

---

### 3. API Key System

* Generate unique API key per user

* Require API key in headers:
  Authorization: Bearer <API_KEY>

* Middleware:

  * Validate API key
  * Track usage
  * Enforce limits

---

### 4. Rate Limiting Logic

* Free → 10 requests lifetime

* Basic → 1,000/month

* Pro → 10,000/month

* Block requests when exceeded

* Reset monthly for paid users

---

### 5. Manual Subscription Flow (MVP)

#### User Side:

#### Admin Side:

* Dashboard view of requests
* Approve/Reject request
* Update user plan manually
* Option to add payment note
